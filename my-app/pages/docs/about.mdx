Welcome to the **Gaming Platform API** documentation.

This API allows you to integrate with our platform for finding tutors and manage various features like user registrations, product catalogs, and order processing.

## Purpose

The purpose of this API is to provide a seamless way to manage tutors, buyers, and sellers on the platform. With this API, you can:
- Add new products
- Process orders
- Handle user registrations
